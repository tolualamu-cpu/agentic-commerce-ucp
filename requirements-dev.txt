# Dev / test-only dependencies (NOT needed to run the app).

# Browser end-to-end tests under tests/browser/ drive a real headless

#            python -m playwright install chromium

#
# The browser tests SKIP automatically (never fail) when Playwright or its
# browser binaries are absent, so `pytest tests/ -x -q` stays green for
# contributors who haven't installed them.
